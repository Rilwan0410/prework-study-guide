Prework Study Guide

HTML

  • The Head element contains information about the webpage.
  • The body element represents the visible content shown to the user

CSS

  • CSS allows us to style the HTML Page.
  • CSS is to HTML what paint is to a house, it is used primarily to style the HTML page to give it life.

GIT

  • Git is a way to control the flow of your project.
  • Git Hub is a version control system that makes it more convenient for collaborations in teams.
  • git status: Checks what branch we are currently working on
  • git checkout -b branch-name: creates a new branch

JavaScript

  • JavaScript is what allows interactivity & functionality to our HTML.
  • JavaScript is to HTML what opening and closing doors, turning on the sink, or turning on a fan is to a house.
